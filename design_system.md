# Design System - Madruga Contabilidade

Based on visual analysis of the reference images.

## 1. Color Palette

### Primary Colors
- **Deep Indigo (Brand Base):** `#232358` (Approximate from hero background)
- **Darker Indigo (Footer/Buttons):** `#151540`
- **Cyan Gradient (Logo/Accents):**
  - Start: `#00D4FF`
  - End: `#0055FF`

### Neutral Colors
- **White:** `#FFFFFF` (Text on dark, card backgrounds)
- **Off-White/Light Grey:** `#F5F7FA` (Page backgrounds)
- **Text Dark:** `#333333` (Headings on light)
- **Text Grey:** `#666666` (Body text)
- **Footer Background:** `#2C2C2C`

### Functional Colors
- **Success (Whatsapp):** `#25D366`
- **Error:** `#FF4444`

## 2. Typography

**Font Family:** Modern Sans-serif (Recommended: `Inter`, `DM Sans`, or `Montserrat`)

### Weights
- **Regular:** 400
- **Medium:** 500
- **Bold:** 700

### Hierarchy
- **H1 (Hero):** 48px/60px, White, Bold
- **H2 (Section Titles):** 36px/44px, Deep Indigo, Bold
- **H3 (Card Titles):** 24px/32px, Deep Indigo, Semi-Bold
- **Body:** 16px/24px, Text Grey, Regular
- **Button Text:** 16px, Semi-Bold

## 3. Spacing & Layout

- **Container Width:** Max 1200px, centered
- **Grid:** 12-column recommended
- **Section Padding:** 80px top/bottom
- **Component Spacing:** 24px (gap between cards)

## 4. Components

### Buttons
- **Primary (Dark):** Background `#151540`, Text `#FFFFFF`, Radius `6px`
- **Secondary (White/Hero):** Background `#FFFFFF`, Text `#151540`, Radius `6px`
- **Hover Effects:** Slight spacing lifting or opacity change

### Cards
- **Background:** White
- **Border Radius:** `8px` to `12px`
- **Shadow:** Soft drop shadow (e.g., `0 4px 20px rgba(0,0,0,0.08)`)
- **Content:** Icon (top), Title, Description

### Form Inputs
- **Background:** White
- **Border:** 1px solid `#E0E0E0`
- **Radius:** `6px`
- **Padding:** 12px 16px

### Icons
- **Style:** Line/Stroke icons (approx 2px stroke width)
- **Color:** Deep Indigo or Black close to text color
